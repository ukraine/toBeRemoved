About the system:

MAD-CMS - a convenient and easy-to-use site management system, that allows a timely creation of very simple websites.

Main features:

  • Creation of unlimited number of pages and sections (does not support multi-tree root yet)
  • Convenient WYSIWYG for novices and HTML editor for advanced users
  • News generation (Unlimited number of news)
  • File management via web interface
  • FAQ generation
  • Forgotten password recovery
  • Feedback form for sending messages directly from the site
  • Automatic displaying of current path of the page
  • Automatic sitemap generation
  • Automatic error message interpretation of non-existing pages
  • Human-friendly URLs (/category/page/ or /category/page.html)

Search engine (SE) compatibility functions:

  • All SEs: Separate fields for keywords, description and the header (title)
  • All SEs: Correct header generation (Last modified date)
  • Google, Yahoo, MSN: Automatic sitemap generation in XML format (Google Sitemaps format)

System requirements

  • Apache version 1.3 and above
  • PHP 4 and above
  • Database MySQL 4 and above
  • Apache mod_rewrite module
